# 哈希表相关理解

## lc.1 两数之和

```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector<int> ret;

        std::unordered_map<int, int> cache;

        for(int i=0;i<nums.size();++i)
        {
            auto it = cache.find(target-nums[i]);
            if(it != cache.end())
            {
                ret.push_back(it->second);
                ret.push_back(i);
                return ret;
            }
            else
            {
                cache[nums[i]] = i;
            }
        }

        return {};
    }
};
```

输入不是严格排序的，所以不能够使用双指针(两数之和II)。

**将每个元素的值和下标，按照key-value塞进map中。**



## lc.49 字母异位词分组

```cpp
class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        std::unordered_map<std::string, std::vector<std::string>> mp;
        
        for(string& str : strs)
        {
            string key = str;
            sort(key.begin(),key.end());
            mp[key].emplace_back(std::move(str));
        }

        std::vector<std::vector<std::string>> ans;
        for(auto& [k,v] : mp)
        {
            ans.emplace_back(std::move(v));
        }

        return ans;
    }
};
```

第一次做时，我直接使用int作为key，没有考虑到ae 和 bb这种的ascii相同。

应该使用string作为key。将字符串sort后就是唯一的顺序。不会产生计数冲突。



## lc.128 最长连续序列

```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        if(nums.size() == 0) return 0;

        unordered_set<int> cache;

        for(int& x : nums)
        {
            cache.insert(x);
        }

        int ans = 0;
        int currNum = 0;
        int currLen = 0;

        for(const int x : cache)
        {
            if(!cache.count(x-1))
            {
                currNum = x;
                currLen = 1;

                while(cache.count(currNum + 1))
                {
                    ++currNum;
                    ++currLen;
                }

                ans = std::max(ans, currLen);
            }
        }

        return ans;
    }
};
```

首先选用unordered_set将nums中的元素去重，且后续查找是O(1)

**接着遍历unordered_set中的元素**

怎么才算**新的**连续序列呢？当某个值-1不存在时，就认为这是一个连续序列的起始元素。当这个值+1，+1...一直存在时，连续序列的长度也自增。

一定会存在多个序列，比如单一的一个元素也算一个序列，**所以需要存在一个持久值记录历史最长序列**

每个新序列结束后，与持久值进行比较，选最大的



## lc.383 赎金信

```cpp
class Solution {
public:
    bool canConstruct(string ransomNote, string magazine) {
        unordered_map<char, int> cache;
        cache.reserve(26);

        for(char& c : magazine)
        {
            ++cache[c];
        }

        for(char& c : ransomNote)
        {
            if(cache[c] != 0) --cache[c];
            else return false;
        }

        return true;
    }
};
```

key：char

value：字符出现的次数

遍历赎金信时，若在哈希表中找得到且value!=0，则有效。若没找到或value==0，直接return false

时间复杂度：O(M+N)

空间复杂度：就26个字母，力扣分析是O(1)



## lc.242 有效的字母异位词

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        if(s.size() != t.size()) return false;

        unordered_map<char, int> cache;

        for(char& c : s)
        {
            ++cache[c];
        }

        for(char& c : t)
        {
            if(cache.count(c) && cache[c] != 0) --cache[c];
            else return false; 
        }

        return true;
    }
};
```

这道题与上一题相同，但需要注意s和t的长度需要一致，否则会出现s=ab    t=a        a满足for中的判断，但t中仅有a，没有b





## lc.205 同构字符串

```cpp
class Solution {
public:
    bool isIsomorphic(string s, string t) {
        unordered_map<char, char> s2t, t2s;

        for(int i=0;i<s.size();++i)
        {
            char a = s[i];
            char b = t[i];

            if(s2t.find(a) != s2t.end() && s2t[a] != b ||
                t2s.find(b) != t2s.end() && t2s[b] != a)
                return false;
            
            s2t[a] = b;
            t2s[b] = a;
        }

        return true;
    }
};
```

**为什么需要两个哈希表？**
s: {b a d c}

t: {b a b a}
当在s2t中没找到字符1时，可能其已经存在了映射关系，而t2s就是避免这种情况的存在

当s2t没找到字符1并且t2s没找到字符2时，说明这是一个新的映射关系

当s2t没找到字符1但t2s找到了字符2时，报错

当s2t找到了字符1但其value不是字符2时，报错

这里有点混乱，但逻辑不复杂，仅有点绕



## lc.290 单词规律

```cpp
class Solution {
public:
    bool wordPattern(string pattern, string s) {
        unordered_map<string, char> str2ch;
        unordered_map<char, string> ch2str;
        int m = s.size();
        int i=0;

        for(auto ch : pattern)
        {
            if(i >= m) return false;

            int j = i;
            while(j < m && s[j] != ' ') j++;
            const string& tmp = s.substr(i, j-i);
            
            if(str2ch.count(tmp) && str2ch[tmp] != ch || ch2str.count(ch) && ch2str[ch] != tmp) return false;

            str2ch[tmp] = ch;
            ch2str[ch] = tmp;

            i = j+1;
        }

        return i >= m;
    }
};
```

这道题与上一道题的区别在于将字符与字符的映射，变成了字符与字符串的映射。 a->cat  b->dog       "abab"  |   "cat dog cat dog"  字符与子字符串一一对应

**难点在于怎么找到一个字符串中的子字符串**

```cpp
int j = i;
while(j < m && s[j] != ' ') j++;	//出现空格时停止
const string& tmp = s.substr(i, j-i);
...
i = j+1;
```

这里就是一个典型的将字符串中非连续的子字符串分割的流程

**保证pattern的长度与s的长度一致**

```cpp
if(i >= m) return false; //防止 pattern 更长、但 s 已经没词了

return i >= m;	//循环结束后，pattern 已处理完。还要确认 s 也恰好处理完，不能有多余单词。
```

**以上两道题需要好好理解，尤其是第二题的长度一致约束**



## lc.202  快乐数

```cpp
class Solution {
public:
    int bitSquareSum(int n) {
        int sum = 0;
        while(n > 0)
        {
            int bit = n % 10;
            sum += bit * bit;
            n = n / 10;
        }
        return sum;
    }
    
    bool isHappy(int n) {
        int slow = n, fast = n;
        do{
            slow = bitSquareSum(slow);
            fast = bitSquareSum(fast);
            fast = bitSquareSum(fast);
        }while(slow != fast);
        
        return slow == 1;
    }
};
```

若最终回到1，则永远是1^2 = 1，存在一个自环。根据环形链表中的floyd算法，快慢指针总是会在这个环中相遇。

**为什么不存在死循环？**

有限状态 + 反复迭代 ⇒ 必然进入环

**会不会出现一个个数>1的环，其中存在1，但是快慢指针在另一个环点相遇？**

如果有1，那这个环只能有1这一个点，因为1^ = 1



## lc.219 存在重复元素

```cpp
class Solution {
public:
    bool containsNearbyDuplicate(vector<int>& nums, int k) {
        unordered_map<int, int> cache;

        for(int i=0;i<nums.size();++i)
        {
            if(cache.count(nums[i]))
                if(std::abs(i - cache[nums[i]]) <= k) return true;

            cache[nums[i]] = i;
        }
        
        return false;
    }
};
```

**这道题可以使用滑动窗口写，空间复杂度降为O(k)**
