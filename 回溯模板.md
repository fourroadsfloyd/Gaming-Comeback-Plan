# 回溯模板

回溯法是一种暴力搜索方法，只不过使用了递归以及某些参数的复用。

1、暴力搜索

2、需要找到结束条件

3、实现处理结点的操作，满足题目返回值要求

4、**保存现场------>恢复现场**     最重要的一点

```cpp
void backtracking(参数) 
{
    if (终止条件) 
    {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) 
    {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
