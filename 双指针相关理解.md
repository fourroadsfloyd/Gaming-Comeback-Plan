# 双指针相关算法

## lc.283 移动零

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int n = nums.size();

        int slow = 0;
        int fast = 0;

        while(fast < n)
        {
            if(nums[fast])
            {
                swap(nums[slow], nums[fast]);
                ++slow;
            }

            ++fast;
        }
    }
};
```

如果数组没有0，那么快慢指针始终指向同一个位置，每个位置自己和自己交换

如果数组有0，快指针先走一步，此时慢指针对应的就是0，所以要交换

慢指针为什么自增之后就一定是0呢？fast走过的都是0，只有停下的不是零。

**只要记好交换操作发生时，++slow即可**



## lc.11 盛水最多的容器

```cpp
class Solution {
public:
    int maxArea(vector<int>& height) {
        int left = 0;
        int right = height.size() - 1;

        int maxArea = INT_MIN;
        int tempArea = 0;

        while(left < right)
        {
            if(height[left] >= height[right])
            {
                tempArea = height[right] * (right - left); 
                --right;
            }    
            else
            {
                tempArea = height[left] * (right - left); 
                ++left;
            }

            maxArea = maxArea < tempArea ? tempArea : maxArea;
        }

        return maxArea;
    }
};
```

这道题的话，仅注意tempArea的求法，高度使用height[left]和height[right]中小的那个。使用谁，谁进行偏移

简单题



## lc.15 三数之和

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;

        sort(nums.begin(), nums.end());

        for(int i=0;i<nums.size();++i)
        {
            if(i > 0 && nums[i] == nums[i-1]) continue;	//第一层去重，保证a不重复

            int l = i+1;
            int r = nums.size()-1;

            while(l < r)
            {
                int sum = nums[i] + nums[l] + nums[r];

                if(sum > 0) --r;
                else if(sum < 0) ++l;
                else
                {
                    ans.emplace_back(vector<int>({nums[i], nums[l], nums[r]}));
					
                    //第二层去重，保证b,c不重复
                    while(l < r && nums[l] == nums[l+1]) ++l;	
                    while(l < r && nums[r] == nums[r-1]) --r;

                    ++l;
                    --r;
                }
            } 
        }

        return ans;
    }
};
```

a+b+c = 0

条件要求这个三元组不能重复，所以代码中**存在两处去重**，保证了每个三元组唯一。

**第一层去重，保证a不重复**

```cpp
if(i > 0 && nums[i] == nums[i-1]) continue;	
```

**第二层去重，保证b,c不重复**

```cpp
while(l < r && nums[l] == nums[l+1]) ++l;	
while(l < r && nums[r] == nums[r-1]) --r;
```



## lc.42 接雨水

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        int left = 0;
        int right = height.size() - 1;
        int leftMax = 0;
        int rightMax = 0;
        int ans = 0;

        while(left < right)
        {
            leftMax = std::max(leftMax, height[left]);
            rightMax = std::max(rightMax, height[right]);

            if(height[left] < height[right])
            {
                ans += leftMax - height[left];
                ++left;
            }
            else
            {
                ans += rightMax - height[right];
                --right;
            }
        }

        return ans;
    }
};
```

**每个位置能接的雨水量 = `min(左侧最大高度, 右侧最大高度) - 当前高度`**

双指针的核心优化：**不需要预先知道另一侧的最大值，只需比较两侧当前最大值的大小来决定移动哪边，谁小谁移动**

| 变量             | 含义                             |
| ---------------- | -------------------------------- |
| `left` / `right` | 左右双指针                       |
| `leftMax`        | `left` 左侧（含）的历史最大高度  |
| `rightMax`       | `right` 右侧（含）的历史最大高度 |
| `ans`            | 累计接水量                       |

---

**逐步解析**

```cpp
leftMax = max(leftMax, height[left]);
rightMax = max(rightMax, height[right]);
```
每次移动前，先更新两侧的历史最大值。

---

```cpp
if (height[left] < height[right]) {
    ans += leftMax - height[left];
    ++left;
}
```
**当左侧高度 < 右侧高度时**，说明右侧一定有一堵墙比 `leftMax` 高（因为 `height[right] > height[left]`），所以 `left` 位置的接水量**只由 `leftMax` 决定**：**（谁小谁决定）**

$$\text{water}[left] = leftMax - height[left]$$

移动左指针向右。

---

```cpp
} else {
    ans += rightMax - height[right];
    --right;
}
```
**当右侧高度 ≤ 左侧高度时**，同理，`right` 位置的接水量**只由 `rightMax` 决定**：

$$\text{water}[right] = rightMax - height[right]$$

移动右指针向左。



## lc.392 判断子序列

```
class Solution {
public:
    bool isSubsequence(string s, string t) {
        int m = s.length();
        int n = t.length();

        int i = 0;
        int j = 0;

        while(i < m && j < n)
        {
            if(s[i] == t[j])
            {
                ++i;
            }
            
            ++j;
        }

        return i==m;
    }
};
```

## lc.167 两数之和 II - 输入有序数组

```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& numbers, int target) {
        int left = 0;
        int right = numbers.size()-1;
        vector<int> ret;
        while(left < right)
        {
            if(numbers[left] + numbers[right] < target)
                ++left;
            else if(numbers[left] + numbers[right] > target)
                --right;
            else
            {
                ret.push_back(left+1);
                ret.push_back(right+1);
                return ret;
            }
        }

        return {};
    }
};
```

题目说 下标从1开始，但是代码中的vector下标总是从0开始，所以只需要在结果处将下标+1即可。
